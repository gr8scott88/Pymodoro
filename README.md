# Pymodoro
Python pomodoro app for Windows

Quick GUI for managing work via pomoro method with the following timing sequeng: 
<br>
	Work: 25 min
	Rest: 5 min 
	Work: 25 min
	Rest: 5 min 
	Work: 25 min
	Rest: 5 min 
	Work: 25 min
	Rest: 15 min 
	
Utilizes "play/pause" keyboard command, so will work with any windows media client.
